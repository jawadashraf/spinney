# Budget Structure: Spinney Hill CRM (Phase 1)

> [!NOTE]
> This structure is based on the 6-week development timeline. Pricing should be adjusted based on your standard day rates.

## 1. Project Phase Breakdown

| Item | Description | Estimated Effort |
| :--- | :--- | :--- |
| **Phase 1: Setup & Foundations** | Env configuration, Auth, RBAC, Tenant logic. | 1.0 Week |
| **Phase 2: Core Development** | Enquiries, Case Files, Notes, Appointments. | 3.0 Weeks |
| **Phase 3: Directory & Donors** | Partners, Outreach, Donation tracking. | 1.0 Week |
| **Phase 4: Reporting & UAT** | Dashboards, Exports, Final QA & Training. | 1.0 Week |
| **TOTAL** | | **6.0 Weeks** |

## 2. Investment Summary

*   **Development & Implementation:** Fixed cost for the 6-week build.
*   **Infrastructure & Hosting:** Monthly subscription for Laravel Herd/Cloud or similar.
*   **Maintenance (Optional):** Retainer for ongoing support and minor updates.

## 3. Resource Allocation
*   **1x Lead Developer:** Architecture, Security, Backend Logic.
*   **1x Product/UI Designer:** Filament Customization, UX Workflow.
*   **1x QA/Project Manager:** Testing, Compliance, Client Communication.

## 4. Why This Approach Wins
*   **Fixed Scope/Fixed Price:** Lowers risk for the client.
*   **Rapid Delivery:** Value is delivered in weeks, not months.
*   **No Technical Debt:** Using Laravel 12 ensures the system is modern for years.
