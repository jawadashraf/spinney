# Project Roadmap & Timeline: Spinney Hill Integrated CRM

This timeline is structured into **6-8 focused weekly sprints**. We have already established core infrastructure and the enquiry intake module.

## 📅 Timeline Summary
| Sprint | Theme | Status | Key Deliverables |
| :--- | :--- | :--- | :--- |
| **Week 1** | **Foundations** | ✅ | Database Schema, Auth, RBAC (Role Based Access Control) |
| **Week 2** | **The Front Door** | ✅ | Enquiry Management & Conversion Flow |
| **Week 3** | **Case Integrity** | 🟡 | Unified Case Files, Restricted Notes, Auth UI Enhancements |
| **Week 4** | **Connect & Care** | 🗓️ | **Care Module**: Session Logs, Timeline, Roles (Check-In/Spiritual/Therapist) |
| **Week 5** | **Management & Engagement** | 🗓️ | **Operations**: Dynamic Forms, Team Mgmt, Tasks, **Payments Module** |
| **Week 6-8** | **Insights & AI Engagement** | 🗓️ | Notifications, **AI Chatbot**, Multi-Role Dashboards, Reporting, Pilot Launch |

---

## 🛠 Sprint Detail (Current & Upcoming)

### Week 3: Service Users & Shared Case Files (In Progress)
*   **Completed:** Integrated Fortify/Filament Auth and configured Auth UI Enhancer.
*   **Completed:** Implemented multi-layout Support (`plain`, `guest2`).
*   **Completed:** Integrated Spatie Activity Log.
*   **Next:** Develop unified Service User profile and restricted session logging architecture.

### Week 4: Care Module & Timeline
*   Implementation of **Session Logs** for Check-In Workers, Spiritual Counselors, and Therapists.
*   Development of the **Shared Timeline** for Service Users (visible roles interaction).
*   **Automated Follow-Up Engine**: Intelligent task generation for Check-In workers (Weekly/Monthly cycles).
*   **Therapist Role**: Elevated permissions to see all notes and suggest "Care Actions".

### Week 5: Operations & Payments
*   **Team Management Module**: Define teams, assign individuals, and track team-level tasks.
*   **Dynamic Scheduled Forms**: Weekly/Daily surveys for Service Users with automated reminder alerts.
*   **Payments Module**: Track charges for therapist sessions (Online, Offline, Bank Transfer).

### Week 6: Notifications & Handover
*   **Multi-Channel Notifications**:
    *   **Live**: Actionable links for urgent tasks.
    *   **Database**: Historical alerts that can be filtered.
    *   **Email**: Targeted communications for roles/teams.
*   Final reporting exports and audit trail for safeguarding.
