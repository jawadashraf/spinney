## üõ† Sprint Detail

### Week 1: Core Foundations
*   Set up infrastructure and high-security tenancy model.
*   Implement User Roles (Frontline, Assessment, Safeguarding, etc.).
*   Deploy Database Models: `Enquiry`, `Appointment`, `EngagementLog`, `SafeguardingFlag`, `Donation`.

### Week 2: Enquiry Management
*   Launch the Enquiry Intake Interface.
*   Implement the "Convert to Service User" workflow.
*   Setup early risk flagging for safeguarding.

### Week 3: Service Users & Shared Case Files
*   Develop the unified Service User profile.
*   Implement standard and "Restricted" note-taking capabilities.
*   Enforce role-based visibility across the platform.

### Week 4: Appointments & Aftercare
*   Shared diary system for booking assessments and sessions.
*   Automatic email confirmation system for bookings.
*   Logging welfare check-ins and follow-ups.

### Week 5: Directories & Donations
*   Partnership directory for faster referrals.
*   Outreach database for school engagement.
*   Donor management system with Gift Aid compliance.

### Week 6: Reporting & Pilot Handover
*   Enable exports for funders and trustees.
*   Finalise the full audit trail for safeguarding compliance.
*   Final QA, documentation, and pilot readiness training.
